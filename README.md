# Telegram Bot with Mistral AI

Легкий Telegram-бот на Python, использующий API Mistral AI для общения с пользователями. Бот умеет поддерживать контекст беседы.

## Технологии
* **Python 3.10+**
* **Aiogram 3.x** (Асинхронная библиотека для Telegram API)
* **MistralAI SDK**
